This is a simple tapestry application to help show project status.

Running Notes

To run in IntelliJ/Eclipse with the local google app engine SDK you need to hack a bit.
Replace appengine-java-sdk-1.3.5/lib/agent/appengine-agent.jar with the one in utils. See   http://dmitrygusev.blogspot.com/2009/08/turn-java-security-manager-off-in.html
for why.
